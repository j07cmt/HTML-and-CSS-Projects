HTML Entities are used to display special characters reserved in HTML. Some characters can confuse the browser, such as the lesser than or greater than character can make the browser incorrectly view them as a tag.
To get around this, we can use an entity name or entity number
For example, I like to visit Japan, so if I wanted to write a blog and use the ¥ symbol, I could write &yen; (entity name) or &#165; (entity number)in my code to tell the browser to display the character without affecting any code.
