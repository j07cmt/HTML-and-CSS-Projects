Basic HTML

Introduction to basic HTML syntax

<article>

The <article> tag is used for a block of content that can be independantly reused. An example of this would be a blog entry, news item, or user-submitted content.

<details> & <summary>

The <details> tag is used to a create a widget, that can be toggled open or closed. The details tag must be used with <summary>, and what ever text is contained in the summary tag will be displayed in the widget when closed.

<time>

The <time> tag is used to present dates and time in a machine readable format. Using the datetime attribute will allow for better search engine results and custom features such as reminders.
